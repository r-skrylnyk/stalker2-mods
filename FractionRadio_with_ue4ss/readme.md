# Fraction Radio

Exploring the Zone with the radio is more atmospheric.

## Installation

Copy files from the archive to:

```
<Game Root>\Stalker2\Binaries\Win64
```

## Compatibility

Game version **1.8.0** and higher.

## Features

- Portable radio that follows you everywhere in the Zone
- Toggle radio on/off with a single hotkey
- Change radio frequencies on the current station
- Hidden and collision-free — won't interfere with gameplay
- **Volume Control** — adjust on the fly; volume persists across on/off toggles
- **8 Quick-Access Stations** — pre-configured hotkeys, customizable in `options.lua`
- Seamless station switching with automatic radio restart
- Optimized asset loading/unloading and better thread safety

## Controls

### Radio

| Hotkey | Action |
|--------|--------|
| Shift + B | Toggle Radio On/Off |
| Shift + F | Change Frequency (random channel on current station) |
| Shift + Page Up | Increase Volume |
| Shift + Page Down | Decrease Volume |

### Quick Station Access

Available Radio Stations (Fraction Radio (second file in downloads))
Mix stations = F1
Neutral = F2
Bandits = F3
Freedom = F4
Military = F5
Duty = F6
Mercenaries = Key.F7
Bonus radio: F9, F10. F12.
