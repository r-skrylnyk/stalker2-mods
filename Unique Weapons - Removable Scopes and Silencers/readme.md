# Unique Weapons — Removable Scopes and Silencers

Allows customization and makes preinstalled scopes/silencers removable on unique weapons. *(beta)*

> **Note:** This mod is currently in testing. Please report any issues — fixes and additions are planned.

## Supported Weapons

**GP37V2** — removable default scope and iron sight; compatible with EN_X4 scope.

**Unique weapons with removable preinstalled attachments:**
Sharpshooter, Cavalier, Trophy, Tank, Sotnyk, Sledgehammer, Texas, Silence, Korshunov PKP, SOFMOD, Lummox.

## Compatibility

- Game version **1.9.0** and higher
- Compatible with any mods that don't overwrite the same `WeaponGeneralSetupPrototypes.cfg` node

## Installation

1. Unzip the archive.
2. Copy `UniqueWeapons_v1.pak` to:
   ```
   S.T.A.L.K.E.R. 2 Heart of Chornobyl\Stalker2\Content\Paks\~mods\
   ```
3. If the `~mods` folder doesn't exist, create it.

## How to Use

Visit a technician and install or remove scopes and silencers on any supported weapon. Previously locked preinstalled attachments are now fully removable.
