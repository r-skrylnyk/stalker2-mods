# Sound Weapons from S.T.A.L.K.E.R.: Shadow of Chernobyl

Replaces default rifle sounds with authentic audio from the original S.T.A.L.K.E.R.: Shadow of Chernobyl, bringing a nostalgic feel to iconic Zone weapons.

## Features

- Replaced single-shot and automatic fire sounds
- Works correctly both indoors and outdoors

## Compatibility

- Tested on Patch **1.8**
- Should not conflict with other mods unless they modify weapon audio files

## Installation

Place the `.pak` file into:

```
Stalker2/Content/Paks/~mods/
```

If the `~mods` folder doesn't exist, create it.
