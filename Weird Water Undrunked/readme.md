This mod removes drunkenness effect while keeping all other artifact stats intact.

COMPATIBILITY:
Game version 1.8.0 and higher.
Compatible with any mods that don't overwrite the same AArtifactWeirdWater node.
Since it uses {bpatch}, it should work even if other mods change the weight or price of the same artifact.

INSTALLATION:
Unzip the archive.
Copy the .pak file to:

S.T.A.L.K.E.R. 2 Heart of Chornobyl\Stalker2\Content\Paks\~mods
If the ~mods folder doesn't exist, create it.
