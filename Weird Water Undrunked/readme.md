# Weird Water Undrunked

Removes the drunkenness effect from the **Weird Water** artifact while keeping all other artifact stats intact.

## Compatibility

- Game version **1.8.0** and higher
- Compatible with any mods that don't overwrite the same `AArtifactWeirdWater` node
- Uses `{bpatch}` — works even if other mods change the weight or price of the same artifact

## Installation

1. Unzip the archive.
2. Copy the `.pak` file to:
   ```
   S.T.A.L.K.E.R. 2 Heart of Chornobyl\Stalker2\Content\Paks\~mods\
   ```
3. If the `~mods` folder doesn't exist, create it.
